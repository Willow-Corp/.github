## 🧠 Overview

**Willow Voice** is an AI-powered voice dictation tool that converts natural speech into polished text across desktop applications. 

It automatically formats punctuation, removes filler words, and adapts writing to your personal style.

The application works with emails, documents, messengers, code editors, and productivity software using simple voice commands.

Ideal for professionals, developers, students, and anyone who wants to write faster using voice.
